# One In The Chamber

## Description

"One In The Chamber" is a fast-paced, top-down shooter game where you face off against waves of enemies. With only one bullet in your chamber at a time, every shot counts. After each shot, you must reload, making timing and accuracy crucial for survival. As you defeat enemies, you'll earn points that can be used to purchase upgrades for your character, such as increased speed, more ammo, and faster bullets.

## Installation

To play "One In The Chamber," you'll need to have Python and Pygame installed on your system.

1.  **Install Python:** If you don't already have Python installed, you can download it from the official website: [https://www.python.org/downloads/](https://www.python.org/downloads/)

2.  **Install Pygame:** Once you have Python installed, you can install Pygame by running the following command in your terminal:

    ```
    pip install pygame
    ```

## How to Play

1.  **Clone the repository:**
    ```
    git clone https://github.com/your-username/one-in-the-chamber.git
    ```
2.  **Navigate to the project directory:**
    ```
    cd one-in-the-chamber
    ```
3.  **Run the game:**
    ```
    python main.py
    ```
## Controls

-   **Movement:**
    -   **W, A, S, D** or **Arrow Keys**: Move the player up, left, down, and right.
-   **Combat:**
    -   **Left Mouse Button**: Shoot
    -   **R**: Reload
    -   **F**: Activate Shield
-   **Game:**
    -   **P**: Pause the game
    -   **F11**: Toggle fullscreen
    -   **+/-**: Zoom in/out
-   **Menu:**
    -   **Arrow Keys**: Navigate the menu
    -   **Enter**: Select an option
    -   **Escape**: Go back to the main menu
